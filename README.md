# liground.github.io
Website of LiGround - A free, open-source and modern Chess Variant Analysis GUI for the 21st century

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Deploy Website
```
npm run deploy
```
